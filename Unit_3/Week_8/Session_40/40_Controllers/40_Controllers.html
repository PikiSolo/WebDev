What is a controller in AngularJS?

Controller is a JavaScript function which is bound to a particular scope.

Controller is the place where we write the business logic.

Controller is created by the developer.

Controller is referred to as $scope in AngularJS.

Example synatx for controller
            var app = angular.module('myApp', []);
            app.controller('myCtrl', function($scope) {
                $scope.firstName = "John";
                $scope.lastName = "Doe";
            });

What is a scope in AngularJS?

Scope is an object that refers to the application model.

Scope is an execution context for expressions.

Scopes are arranged in hierarchical structure which mimic the DOM structure of the application.

Scopes can watch expressions and propagate events.

Scopes are objects that refer to the model.

Scopes can be nested to limit access to the properties of application.

Scope is referred to as $scope in AngularJS.
