What is directive in AngularJS?

Directives are markers on DOM elements (such as elements, attributes, css, and more). These can be used to create custom HTML tags that serve as new, custom widgets. AngularJS has built-in directives (ngBind, ngModel...) to perform most of the tasks.

AngularJS directives are extended HTML attributes with the prefix ng-.

List of Commonly used Directives in AngularJS

  • ng-app

    • Defines the root element of an AngularJS application, and it is typically placed near the root element of the page - e.g. on the 'html' or 'body'tags.

  • ng-controller

    • Defines the application controller class.

  • ng-init

    • ng-init helps to initialization values for an application.

  • ng-model

    • ng-model binds the value of HTML controls (input, select, textarea) to application data.

  • ng-bind

    • ng-bind binds the innerHTML of the element to application data.

  • ng-repeat

    • ng-repeat repeats an HTML element for each item in an array or an object.

  • ng-if

    • ng-if removes the HTML element if the expression evaluates to false.

  • ng-switch

    • ng-switch is used to conditionally swap DOM structure on your template based on a scope expression.

  • ng-show

    • ng-show shows the HTML element if the expression evaluates to true.

  • ng-hide

    • ng-hide hides the HTML element if the expression evaluates to true.

  • ng-click

    • ng-click allows you to specify custom behavior when an element is clicked.

  • ng-style

    • ng-style allows you to set CSS style on an HTML element conditionally.
