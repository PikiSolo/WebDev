List of commonly used directives in AngularJS

  • ng-app

    • What is ng-app?

    ng-app directive is used to define the root element of an AngularJS application. It is used to auto-bootstrap an AngularJS application. AngularJS application cannot be started without using this directive.

  • ng-model

    • What is ng-model?

    ng-model directive is used to bind the value of HTML controls (input, select, textarea) to application data.

  • ng-bind

    • ng-bind directive is used to bind the innerHTML of an element to application data.
