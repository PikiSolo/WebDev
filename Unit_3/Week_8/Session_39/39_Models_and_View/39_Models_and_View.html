What is a model in AngularJS?

Model is the data that is used to display the view.

Model is just a JavaScript object which holds the data and these data are displayed in the view.

Model is created by the controller.

Model is referred to as $scope in AngularJS.

Example synatx for model
            var app = angular.module('myApp', []);
            app.controller('myCtrl', function($scope) {
                $scope.firstName = "Guru";
                $scope.lastName = "J";
            });

What are views in AngularJS?

Views are the HTML that is rendered in the browser.

Example synatx for view
            

                First Name: 

                Last Name: 

                

                Full Name: {{firstName + " " + lastName}}
