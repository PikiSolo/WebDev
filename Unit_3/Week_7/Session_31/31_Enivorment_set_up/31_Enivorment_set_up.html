What is ng-app?

ng-app is a directive that tells AngularJS that the element is the "owner" of an AngularJS application.

What are {{}} in angularjs?

{{ }} are called expressions. They are used to bind application data to html.

What is ng-controller?

ng-controller is a directive that attaches a controller class to the view. This is a key aspect of how angular supports the principles behind the Model-View-Controller design pattern.

What is $scope?

&scope is a javascript object that is used to pass data between controller and view.
