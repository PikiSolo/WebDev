String in JavaScript

What is String?

A string is a sequence of one or more characters that may consist of letters, numbers, or symbols.

What is String Literals?

String literals are enclosed in single or double quotes. For example:

var str1 = "Hello World!";

var str2 = 'Hello World!';

String Methods

String methods help you to work with strings.

  • charAt()

    • The charAt() method returns the character at the specified index in a string.

    Syntax of charAt()

    string.charAt(index)
    var str = "Hello World!";
    var res = str.charAt(0);
  • charCodeAt()

    • The charCodeAt() method returns the Unicode of the character at the specified index in a string.

    Syntax of charCodeAt()

    string.charCodeAt(index)
    var str = "Hello World!";
    var res = str.charCodeAt(0);
  • concat()

    • The concat() method joins two or more strings.

    Syntax of concat()

    string.concat(string1, string2, ..., stringX)
    var str1 = "Hello ";
    var str2 = "World!";
    var res = str1.concat(str2);
  • indexOf()

    • The indexOf() method returns the index of (the position of) the first occurrence of a specified text in a string.

    Syntax of indexOf()

    string.indexOf(searchvalue, start)
    var str = "Hello world, welcome to the universe."; 
    var n = str.indexOf("welcome");
  • lastIndexOf()

    • The lastIndexOf() method returns the index of the last occurrence of a specified text in a string.

    Syntax of lastIndexOf()

    string.lastIndexOf(searchvalue, start)
    var str = "Hello planet earth, you are a great planet."; 
    var n = str.lastIndexOf("planet");
  • localeCompare()

    • The localeCompare() method compares two strings in the current locale.

    Syntax of localeCompare()

    string.localeCompare(compareString)
    var str1 = "ab";
    var str2 = "cd";
    var n = str1.localeCompare(str2);
  • match()

    • The match() method searches a string for a match against a regular expression, and returns the matches, as an Array object.

    Syntax of match()

    string.match(regexp)
    var str = "The rain in SPAIN stays mainly in the plain";
    var res = str.match(/ain/g);
  • replace()

    • The replace() method searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced.

    Syntax of replace()

    string.replace(searchvalue, newvalue)
    var str = "Visit Microsoft!";
    var res = str.replace("Microsoft", "W3Schools");
  • search()

    • The search() method searches a string for a specified value, and returns the position of the match.

    Syntax of search()

    string.search(searchvalue)
    var str = "Visit W3Schools!";
    var n = str.search("W3Schools");
  • slice()

    • The slice() method extracts parts of a string and returns the extracted parts in a new string.

    Syntax of slice()

    string.slice(start, end)
    var str = "Apple, Banana, Kiwi";
    var res = str.slice(7, 13);
  • split()

    • The split() method is used to split a string into an array of substrings, and returns the new array.

    Syntax of split()

    string.split(separator, limit)
    var str = "How are you doing today?";
    var res = str.split(" ");
  • substr()

    • The substr() method extracts parts of a string, beginning at the character at the specified position, and returns the specified number of characters.

    Syntax of substr()

    string.substr(start, length)
    var str = "Hello world!";
    var res = str.substr(1, 4);
  • substring()

    • The substring() method extracts the characters from a string, between two specified indices, and returns the new sub string.

    Syntax of substring()

    string.substring(start, end)
    var str = "Hello world!";
    var res = str.substring(1, 4);
  • toLocaleLowerCase()

    • The toLocaleLowerCase() method converts a string to lowercase letters, according to the host's locale.

    Syntax of toLocaleLowerCase()

    string.toLocaleLowerCase()
    var str = "Hello World!";
    var res = str.toLocaleLowerCase();
  • toLocaleUpperCase()

    • The toLocaleUpperCase() method converts a string to uppercase letters, according to the host's locale.

    Syntax of toLocaleUpperCase()

    string.toLocaleUpperCase()
    var str = "Hello World!";
    var res = str.toLocaleUpperCase();
  • toLowerCase()

    • The toLowerCase() method converts a string to lowercase letters.

    Syntax of toLowerCase()

    string.toLowerCase()
    var str = "Hello World!";
    var res = str.toLowerCase();
  • toUpperCase()

    • The toUpperCase() method converts a string to uppercase letters.

    Syntax of toUpperCase()

    string.toUpperCase()
    var str = "Hello World!";
    var res = str.toUpperCase();
  • toString()

    • The toString() method returns the value of a string.

    Syntax of toString()

    string.toString()
    var str = "Hello World!";
    var res = str.toString();
  • trim()

    • The trim() method removes whitespace from both sides of a string.

    Syntax of trim()

    string.trim()
    var str = "       Hello World!        ";
    var res = str.trim();
  • valueOf()

    • The valueOf() method returns the primitive value of a string object.

    Syntax of valueOf()

    string.valueOf()
    var str = "Hello World!";
    var res = str.valueOf();
