What is Keyword

Keyword is a reserved word in JavaScript. It has a special meaning in JavaScript. We can not use keyword as a variable name, function name or any identifier name.

List of Keywords

await,break,case,catch,class,const,continue,debugger,default,delete,do,else,enum,export,extends,false,finally,for,function,if,implements,import,in,instanceof,interface,let,new,null,package,private,protected,public,return,super,switch,static,this,throw,try,true,typeof,var,void,while,with,yield

  • await : The await operator is used to wait for a Promise. It can only be used inside an async function.
  • break : The break statement "jumps out" of a loop.
  • case : The case statement is used to match a value to a condition.
  • catch : The catch statement allows you to define a block of code to be executed, if an error occurs in the try block.
  • class : The class keyword declares a new class.
  • const : The const keyword declares a constant variable, which cannot be re-assigned a new value.
  • continue : The continue statement "jumps over" one iteration in the loop.
  • debugger : The debugger keyword stops the execution of JavaScript, and calls (if available) the debugging function.
  • default : The default keyword specifies the code to run if there is no case match in switch statement.
  • delete : The delete keyword deletes a property from an object.
  • do : The do/while statement creates a loop that executes a block of code once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.
  • else : The else statement specifies a block of code to be executed if the condition is false.
  • enum : The enum keyword is used to declare an enumeration (a set of named constants).
  • export : The export keyword is used when creating JavaScript modules to export functions, objects, or primitive values from the module so they can be used by other programs with the import keyword.
  • extends : The extends keyword is used to create a class that is a child of another class (parent).
  • false : The false keyword represents a false value.
  • finally : The finally statement lets you execute code, after try and catch, regardless of the result.
  • for : The for statement creates a loop that is executed as long as a condition is true.
  • function : The function keyword declares a function expression.
  • if : The if statement specifies a block of code to be executed if a condition is true.
  • implements : The implements keyword is used to implement an interface.
  • import : The import keyword is used to import functions, objects, or primitives that have been exported from an external module, another script, etc.
  • in : The in keyword returns true if the specified property is in the specified object.
  • instanceof : The instanceof keyword returns true if an object is an instance of an object type.
  • interface : The interface keyword is used to declare a special type of object.
  • let : The let keyword declares a block scope local variable, optionally initializing it to a value.
  • new : The new keyword is used to create new objects.
  • null : The null keyword represents a null value.
  • package : The package keyword declares a package.
  • private : The private keyword is used to declare a private property or method.
  • protected : The protected keyword is used to declare a protected property or method.
  • public : The public keyword is used to declare a public property or method.
  • return : The return keyword stops the execution of a function and returns a value from that function.
  • super : The super keyword refers to the parent class.
  • switch : The switch statement is used to perform different actions based on different conditions.
  • static : The static keyword defines a static method for a class.
  • this : The this keyword refers to the current object.
  • throw : The throw keyword throws (generates) an error.
  • try : The try statement allows you to define a block of code to be tested for errors while it is being executed.
  • true : The true keyword represents a true value.
  • typeof : The typeof keyword returns the type of a variable, object, function or expression.
  • var : The var keyword declares a variable.
  • void : The void keyword discards a return value from a function.
  • while : The while statement creates a loop that is executed while a specified condition is true.
  • with : The with keyword lets you access the properties of an object without having to specify the object's name.
  • yield : The yield keyword is used to pause and resume a generator function (function* or legacy generator function).
