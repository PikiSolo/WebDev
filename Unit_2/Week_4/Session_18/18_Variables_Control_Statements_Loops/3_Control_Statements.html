What is control statements in Javascript?

Control statements are used to control the flow of execution of a program based on certain conditions.
JavaScript supports the following control statements:

  • Conditional Statements
  • Looping Statements
  • Jump Statements

What is conditional statements in Javascript?

Conditional statements are used to perform different actions based on different conditions.
JavaScript supports the following conditional statements:

  • if...else
  • switch

What is if...else statement in Javascript?

The if...else statement is used to execute a block of code based on the condition.
If the condition evaluates to true, then the if block of code will be executed, otherwise else block of code will be executed.

Example for if else

synatax of if..else

        if (condition) {
            // block of code to be executed if the condition is true
        } else {
            // block of code to be executed if the condition is false
        }

What is switch statement in Javascript?

The switch statement is used to execute one of the many blocks of code based on the expression.

Example for switch

synatax of switch

        switch (expression) {
            case value1:
                // block of code to be executed if expression = value1
                break;
            case value2:
                // block of code to be executed if expression = value2
                break;
            ...
            case valueN:
                // block of code to be executed if expression = valueN
                break;
            default:
                // block of code to be executed if expression is different from all values
        }

What is looping statements in Javascript?

Looping statements are used to execute a block of code repeatedly as long as the condition is true.
JavaScript supports the following looping statements:

  • for
  • while
  • do...while

What is for loop in Javascript?

The for loop is used to execute a block of code for a specified number of times.

Example for for loop

synatax of for loop

        for (initialization; condition; increment/decrement) {
            // block of code to be executed
        }

What is while loop in Javascript?

The while loop is used to execute a block of code as long as the condition is true.

Example for while loop

synatax of while loop

        while (condition) {
            // block of code to be executed
        }

What is do...while loop in Javascript?

The do...while loop is used to execute a block of code as long as the condition is true.
The block of code will be executed once before the condition is checked.

Example for do...while loop

synatax of do...while loop

        do {
            // block of code to be executed
        } while (condition);

What is jump statements in Javascript?

Jump statements are used to transfer the control of the program to a specific statement.
JavaScript supports the following jump statements:

  • break
  • continue

What is break statement in Javascript?

The break statement is used to terminate the loop or switch statement and transfer the control over the next statement.

Example for break statement

synatax of break statement

        break;

What is continue statement in Javascript?

The continue statement is used to terminate the current iteration of the loop and continue with the next one.

Example for continue statement

synatax of continue statement

        continue;
