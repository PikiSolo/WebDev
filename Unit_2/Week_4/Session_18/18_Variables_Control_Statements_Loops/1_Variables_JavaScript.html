Variables in JavaScript

Variables in JavaScript are containers to store data values.

JavaScript uses reserved keyword var to declare a variable.

Two type of variables in Javascript are local and global.

Local Variables

Variables declared within a JavaScript function, become local to the function.

Global Variables

Variables declared outside of any function, become global.

Point to note : JS variable are lossely typed. It means that a variable can hold any type of value.

What are Strings?

Strings are used to store text.

Strings are written inside double or single quotes.

Example

What are Numbers?

Numbers are written without quotes.

If you put quotes around a number, it will be treated as a text string.

What are Booleans?

Booleans can only have two values: true or false.

What are Objects?

Objects are variables too. But objects can contain many values.

What are Functions?

Functions are also variables. But functions can contain code lines.

Functions are executed when they are called. This is known as invoking a function.

What are Arrays?

Arrays are special types of objects.

Arrays are used to store multiple values in a single variable.

What are Null?

Null is a special type of data type which can have only one value : null.

What are Undefined?

Undefined is a type of data type which can have only one value : undefined.
