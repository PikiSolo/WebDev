Operators in JavaScript

What are Operators in JavaScript?

Operators are used to perform specific mathematical and logical computations on operands.
JavaScript supports the following types of operators:

  • Arithmetic Operators
  • Comparison Operators
  • Logical Operators
  • Assignment Operators
  • Conditional Operators

Arithmetic Operators

What are Arithmetic Operators?

Arithmetic operators are used to perform arithmetic operations on operands.
The following table lists the arithmetic operators supported by JavaScript:

Operator Description
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus
++ Increment
-- Decrement

Comparison Operators

What are Comparison Operators?

Comparison operators are used to compare two operands.
The following table lists the comparison operators supported by JavaScript:

Operator Description
== Equal to
=== Equal value and equal type
!= Not equal
!== Not equal value or not equal type
> Greater than
< Less than
>= Greater than or equal to
<= Less than or equal to

Logical Operators

What are Logical Operators?

Logical operators are used to determine the logic between variables or values.
The following table lists the logical operators supported by JavaScript:

Operator Description
&& Logical and
|| Logical or
! Logical not

Assignment Operators

What are Assignment Operators?

Assignment operators are used to assign values to JavaScript variables.
The following table lists the assignment operators supported by JavaScript:

Operator Example Same As
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y

Conditional Operators

What are Conditional Operators?

Conditional operators are used to evaluate a condition that has two operands and returns true or false.
The following table lists the conditional operators supported by JavaScript:

Operator Description
? condition ? val1 : val2
