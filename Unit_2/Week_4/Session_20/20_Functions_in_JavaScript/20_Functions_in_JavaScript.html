Functions in JavaScript

What are functions in Javascript?

Functions are a way of packaging functionality that you wish to reuse. Functions are also a way of packaging functionality that you wish to run at a later time or in response to an event.

How to define a function in JavaScript?

There are two ways to define a function in JavaScript:

  • Function Declaration
  • Function Expression

Function Declaration

Function declarations are hoisted to the top of the enclosing scope. This means that you can call a function before it is declared:

        
            sayHello();
            function sayHello() {
                console.log("Hello!");
            }

Function declarations are not terminated with a semicolon.

Function Expression

Function expressions are not hoisted to the top of the enclosing scope. This means that you cannot call a function expression before it is declared:

        
            sayHello();
            var sayHello = function() {
                console.log("Hello!");
            };

What is the difference between function declaration and function expression?

Function declarations are hoisted to the top of the enclosing scope. This means that you can call a function before it is declared. Function expressions are not hoisted to the top of the enclosing scope. This means that you cannot call a function expression before it is declared.

Various ways to invoke function in javascript

  • Function Invocation

    • Function invocation is the most common way to invoke a function. It involves invoking a function by using the function name followed by parentheses. If the function has parameters, you pass the arguments into the parentheses.

                
                function sayHello(name) {
                    console.log("Hello " + name);
                }
                sayHello("John");
  • Method Invocation

    • Method invocation is similar to function invocation. The only difference is that you invoke a method on an object. The method invocation is also known as a method call.

                    
                    var person = {
                        name: "John",
                        sayHello: function() {
                            document.write("Hello " + this.name);
                        }
                    };
                    person.sayHello();
  • Constructor Invocation

    • Constructor invocation is used to create a new object. The new operator creates an empty object and then invokes the function specified after the new operator. The function invoked by the new operator is called a constructor function. The constructor function initializes the properties of the new object.

                    
                    function Person(name) {
                        this.name = name;
                    }
                    var person = new Person("John");
                    document.write(person.name);
  • Apply Invocation

    • Apply invocation is similar to function invocation. The only difference is that you can pass an array of arguments to the function.

                    
                    function sayHello(name) {
                        document.write("Hello " + name);
                    }
                    sayHello.apply(null, ["John"]);
