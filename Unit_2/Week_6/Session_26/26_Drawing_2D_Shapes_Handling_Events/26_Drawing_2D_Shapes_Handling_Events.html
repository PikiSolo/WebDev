What is canvas ?

Canvas is an HTML element which can be used to draw graphics using JavaScript.

What is canvas.getContext()?

getContext() method returns an object which provides methods and properties for drawing on the canvas.

What is beginPath() ?

beginPath() method starts a path or resets the current path.

What is arc() ?

arc() method creates an arc/curve. It takes 6 parameters: x, y, radius, startAngle, endAngle, anticlockwise

What other methods apart from arc() can be used to draw shapes ?

moveTo() , lineTo() , stroke() and etc

What is stroke() ?

stroke() method actually draws the path you have defined with all those moveTo() and lineTo() methods. The default color is black.

What is fill()?

fill() method fills the current drawing (path). The default color is black.

what is fillStyle() ?

fillStyle() method sets or returns the color, gradient, or pattern used to fill the drawing.
