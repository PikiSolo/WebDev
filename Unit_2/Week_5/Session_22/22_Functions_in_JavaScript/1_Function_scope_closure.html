Scope:

Scope determines the accessibility (visibility) of variables.

Two types of scope in JavaScript:

1. Local scope

Local scope is created when a function starts, and deleted when the function is completed.

2. Global scope

Global scope is created when JavaScript code is executed, and deleted when the browser window is closed.

Closure

A closure is a function having access to the parent scope, even after the parent function has closed.

This means a closure can remember and access variables and arguments of its outer function even after the function has finished.
