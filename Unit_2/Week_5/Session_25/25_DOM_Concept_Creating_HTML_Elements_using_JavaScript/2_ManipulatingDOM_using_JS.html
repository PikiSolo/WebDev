What is DOM manipulation?

DOM manipulation is the process of changing the DOM using JavaScript. It is used to add, delete, update and style HTML elements.

What are NODE

Node in DOM is an object. It is used to represent HTML elements. It can be used to create, delete and modify HTML elements.
