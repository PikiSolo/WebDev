Different methods to access the form elements

There are different methods to access the form elements

  • getElementById

    • What is getElementById? getElementById is a method which is used to access the form elements by using the id of the element.

  • getElementsByName

    • What is getElementsByName? getElementsByName is a method which is used to access the form elements by using the name of the element.

  • getElementsByTagName

    • What is getElementsByTagName? getElementsByTagName is a method which is used to access the form elements by using the tag name of the element.
