What is a HTML form?

An HTML form is a section of a document containing normal content, markup, special elements called controls (checkboxes, radio buttons, menus, etc.), and labels on those controls. Users generally "complete" a form by modifying its controls (entering text, selecting menu items, etc.), before submitting the form to an agent for processing (e.g., to a Web server, to a mail server, etc.)

What are the various Form elements

  • Text Input

    • What is text Input?

    Text input is a common element of forms. For example, when you want to fill out a form on a website, you click into the text input field and type in the information that you want to enter. The text input field is one line high, by default. You can change the height and width of the text input field by using the size attribute. The size attribute specifies the visible width, in characters, of an input element.

  • Radio Button

    • What is Radio Button?

    Radio buttons are groups of buttons in which, by convention, only one button at a time can be selected. The name attribute is used to identify the group of radio buttons. The value attribute is used to specify the value that is sent to the server when the form is submitted. The checked attribute is used to specify that a radio button is selected by default..

    Checkbox

    What is Checkbox?

    Checkboxes are groups of buttons in which, by convention, any number of buttons at a time can be selected. The name attribute is used to identify the group of checkboxes. The value attribute is used to specify the value that is sent to the server when the form is submitted. The checked attribute is used to specify that a checkbox is selected by default.

  • Dropdown

    • What is Dropdown?

    Dropdowns are lists of options that appear when the user clicks on a button or text selection. The select element is used to create a dropdown list. The option element is used to define the available options in a dropdown list. The selected attribute is used to specify which option is selected by default when the page loads.

  • Textarea

    • What is Textarea?

    Textarea is used to create a multi-line text input. The cols attribute specifies the visible width of a text area. The rows attribute specifies the visible number of lines in a text area.

  • Button

    • What is Button?

    Buttons are used to submit forms. The type attribute is used to specify the type of button. The value attribute is used to specify the text that appears on the button.

  • Submit

    • What is Submit?

    Submit buttons are used to submit forms. The type attribute is used to specify the type of button. The value attribute is used to specify the text that appears on the button.

  • Reset

    • What is Reset?

    Reset buttons are used to reset forms. The type attribute is used to specify the type of button. The value attribute is used to specify the text that appears on the button.
