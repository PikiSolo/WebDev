In this example we are going to see how onclick will change the text input color and submit the form value to the broswer>


Various methods for FORM handling , elements and form output in Javascript

  • onsubmit

    • The onsubmit event occurs when a form is submitted. It only applies to the FORM element. The onsubmit event is not triggered if you call the form.submit() method.

  • onreset

    • The onreset event occurs when a form is reset. It only applies to the FORM element.

  • onclick

    • onclick is used to perform some action when an element is clicked.

  • onchange

    • onchange is used to perform some actions when the user changes the value of an element

  • onfocus

    • onfocus is used to perform some actions when an element gets focus

  • onblur

    • onblur is used to perform some actions when an element loses focus

  • onkeydown

    • onkeydown is used to perform some actions when a key is pressed down over an element

  • onkeypress

    • onkeypress is used to perform some action when the the key is pressed and released

  • onkeyup

    • onkeyup is used to perform some action when the the key is released

    various methods for form output
