What is function parameter?

A function parameter is a variable that is used to define the data that is passed into a function. 

Syntax to function parameter
            function functionName(parameters) {
                // code to be executed
                // return statement
            }

What is function argument?

A function argument is a value that is passed into a function when the function is called. 

Syntax to function argument
            functionName(arguments);

What is function return?

A function return is a value that is returned from a function when the function is called. 

Syntax for function return
            function functionName(parameters) {
                // code to be executed
                // return statement
            }
