Modifying DOM object

What is a DOM?

DOM stands for Document Object Model. Its platform and language neutral. It is used to represent and interact with HTML, XHTML and XML documents. It is a tree structure of nodes and objects.

How are DOM object modified?

  • document.getElementById(id) - Finds an element by element id
  • document.getElementsByTagName(name) - Finds an element by element tag name
  • document.getElementsByClassName(name) - Find elements by class name
